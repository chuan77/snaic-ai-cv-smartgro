# Project Handoff — 2026-07-07

## What This Project Does

SmartCart AI Checkout Assistant: an autonomous retail checkout system built as a 5-stage pipeline (Python/Ultralytics YOLO11/DINOv2) that turns unannotated grocery product photos into a working object detector, plus a React/Vite frontend and (as of today) a FastAPI backend that serves real detections and catalog pricing to that frontend. Stack: Python >=3.14 (`uv`), PyTorch/Ultralytics YOLO11, FastAPI/uvicorn, React/TypeScript/Vite, pytest. Runs on Apple Silicon (`mps` device).

**Note on timing:** git history shows a `subagent-driven-development` plan run was still actively committing while this doc was being written (HEAD moved from `d25370b` to `586e6f7` to `43f1451` within minutes). Everything below reflects the state as of commit `43f1451`, `git status` at that point, and a passing local test run — check `git log -1` before trusting exact task-completion claims.

## Running the Stack

```bash
# Install deps
uv sync

# Run the ML pipeline (in order, from repo root)
python main_day1_catalog.py      # build catalog + DINOv2 gallery
python main_day2_train.py        # synthesize scenes + train YOLO (5 classes)
python main_day3_fine_tune.py    # evaluate against synthetic_dataset/data.yaml
python main_day4_optimize.py     # shadow-stress augmentation for weak_class_id=4
python main_day5_deploy.py       # Gradio dashboard (standalone deployment path)

# Run tests
uv run pytest -q                 # 42 passed, 1 deselected (slow marker)
uv run pytest -m slow -v         # real-inference smoke test against yolo11n.pt

# Run the API backend + frontend (once Task 6 lands)
uv run python main_api_server.py       # http://0.0.0.0:8000
cd frontend && npm install && npm run dev   # http://localhost:5173
```
